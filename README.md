# Car Browsing Site

## Overview
The Car Browsing Site is a web application designed to help users browse, search, and filter through a wide range of cars using HTML, CSS, and JavaScript.

## features:
- **Navigation Bar** :  with options such as About, New Arrivals, FAQs, and Search by Brands.
- **Car Details Page** : Implemented a feature where users can scroll through images of branded cars and click on them to navigate to pages displaying respective car models and their prices.

## Technologies Used :
- **Frontend**: HTML, CSS, JavaScript
